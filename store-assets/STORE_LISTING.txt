КарПас — графика для публикации в магазинах
============================================

Сгенерированные PNG и скрипт npm run store:assets удалены: кладите сюда свои
файлы вручную (дизайнер, экспорт из Figma и т.д.).

Рекомендуемые папки:
  store-assets/google-play/   — иконка 512x512, баннер 1024x500, скриншоты
  store-assets/app-store/     — иконка 1024x1024, скриншоты для App Store Connect
  store-assets/source/        — по желанию: ваши исходники (SVG, PSD)

Иконки самого приложения на Android по-прежнему в:
  android/app/src/main/res/mipmap-*/
