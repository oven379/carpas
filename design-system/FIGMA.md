# Дизайн-система КарПас → Figma

В коде нет отдельного `.fig` файла: стили заданы в **CSS** (`src/index.css`, `src/App.css`) и **Tailwind** (`tailwind.config.js`). Здесь — **токены** для переноса в Figma.

## Что лежит в папке

| Файл | Назначение |
|------|------------|
| `tokens/carpas.tokens.json` | Полный набор + комментарии (W3C DTCG, `$value` / `$type`) |
| `tokens/carpas.tokens-studio.json` | Упрощённый набор для импорта в **Tokens Studio** (`value` / `type`) |

## Как открыть в Figma

1. Установите плагин **[Tokens Studio for Figma](https://tokens.studio/)** (или **Figma Tokens**).
2. В Figma: **Plugins → Tokens Studio → Load from file / Import**.
3. Сначала попробуйте **`carpas.tokens-studio.json`**; если плагин просит DTCG — **`carpas.tokens.json`**.
4. Примените токены к **Variables** (в Tokens Studio: **Styles & Variables** / экспорт в нативные переменные Figma — зависит от версии плагина).

Если плагин ругается на `$schema` или вложенность, откройте JSON и временно удалите ключ `"$schema"` и при необходимости упростите вложенные объекты до плоского списка групп (плагины отличаются).

## Адаптив: веб и приложение (единая вёрстка)

Один и тот же UI в **десктопном браузере**, **мобильном браузере** и **Capacitor (Android/iOS WebView)**. Стили — `src/App.css`, `src/index.css`; в WebView используется тот же бандл, что и на сайте.

| Условие | Где в `App.css` / `index.css` | Назначение |
|--------|-------------------------------|------------|
| `max-width: 1024px` | `index.css` `:root` | Базовый кегль 16px вместо 18px |
| `max-width: 980px` | `App.css` | Сетки лендинга/форм в одну колонку и др. |
| `max-width: 768px` | `App.css` | `.main` с отступом снизу под **safe-area**; `.container` 12px + `env(safe-area-inset-*)` |
| `max-width: 720px` | `App.css` | Герои карточек, гаражный баннер, компактные отступы |
| `max-width: 640px` | `App.css` | Шапка навигации с safe-area сверху/сбоку; блок **профиля гаража** (см. ниже) |
| `max-width: 360px` | `App.css` | Доп. уплотнение текста под аватаром на очень узких экранах |

**Safe area (вырез, индикатор «домой»):** у `.container` горизонтальные отступы уже учитывают `env(safe-area-inset-left/right)`; у `.main` — нижний inset. В шапке `.nav__inner` на узкой ширине добавлен **padding-top** с `env(safe-area-inset-top)`.

## Экран «Гараж владельца» (`/garage`, `OwnerGaragePage.jsx`)

Макет в Figma: фреймы **Desktop** (ширина контента до **1126px**, класс `.container`) и **Mobile** (320–430px).

1. **Баннер** (если включён в настройках) — `.garagePage .garageHero`, высота ~300px (десктоп), ~240px при `max-width: 720px`. Кнопка настроек в углу оверлея.
2. **Карточка профиля** — `.garageProfileCard`:
   - **Верхний ряд** — CSS Grid: основной блок слева (`Гараж`, имя, город, иконки контактов, последний визит), справа колонка **аватар** (ссылка в настройки) в **правом верхнем углу** карточки, под ним подсказка **«Нажмите на аватар для настройки страницы.»** (выравнивание текста вправо, `max-width` ограничивает переносы на мобилке).
   - **Город:** если не задан — приглушённый текст **«нет данных»** (без тире и без отдельной ссылки «настройки»; настройка через аватар).
   - **Последний визит:** при отсутствии событий — фраза **«Пока нет истории визитов»** (тот же смысл на карточке авто в `CarPage.jsx`).
3. **Нижний ряд карточки** — логотип привязанного сервиса (если есть), кнопка **«Добавить авто»**.

Классы для стилизации в Figma: `garageProfileCard__top`, `garageProfileCard__main`, `garageProfileCard__avatarCol`, `garageProfileCard__avatarHint`, `garageProfileCard__cityLine`, `garageProfileCard__lastVisit*`.

## Поведение и производительность (для согласования с продуктом)

- Список авто и блок «Последний визит» используют **одну** загрузку событий по машинам (без второго раунда запросов к API).
- Страница гаража не блокируется полноэкранным спиннером после ответа списка машин: компактный индикатор только у списка, пока подгружаются события.

## Ограничения

- Часть цветов задана через **`color-mix(in oklab, …)`** — в Figma нет прямого аналога; в JSON даны **ближайшие HEX/RGB** или подпись «задайте вручную».
- Тёмная тема **одна** (`color-scheme: dark`), `prefers-color-scheme` не переключает тему.

## Шрифт

В проде ориентир: **Roboto** + системный стек. В Figma подключите **Roboto** (Google Fonts) и переменные семейства из токенов.
